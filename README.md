# 📢 ZYXCAD AX 제안서 사용 안내

> 블루티엔에스 영업팀용 · ZYXCAD AX / ZYX SPACE 제안서 (HTML 슬라이드 22장)
> 고객사 정보만 입력하면 표지·연락처가 자동으로 바뀌고, 바로 PDF로 저장할 수 있습니다.

---

## 1. 파일 열기

- `index.html` 파일을 **더블클릭**하면 브라우저(Chrome / Edge 권장)에서 바로 열립니다.
- 좌우 방향키 또는 화면 하단 썸네일로 슬라이드를 이동합니다.
- ⚠️ `index.html` · `assets` 폴더 · `deck-stage.js` · `support.js`는 **같은 위치**에 함께 있어야 합니다.

---

## 2. 고객사 맞춤 입력 (좌측 ⚙ 설정)

화면 좌측 상단 **⚙ 설정** 버튼을 누르면 입력 패널이 열립니다.

### A. 고객사
- **고객사명** — 입력하면 표지의 「〇〇 귀중」과 PDF 제목에 자동 반영
- **고객사 CI 업로드** — **배경이 투명한 PNG** 권장. 표지에 흰 박스 없이 투명하게 삽입
  - 어두운 표지 배경에는 **흰색/밝은색 로고**가 가장 선명합니다
  - **로고 크기 (비율)** 슬라이더로 50%~200% 조절, 가운데 `100%` 버튼으로 기본값 복원
- **제안 일자 / 문서번호** — 선택 입력

### B. 제안 담당자 (블루티엔에스)
- 성함 · 직함 · 부서 · 연락처 · 이메일 입력
- → **기술지원(12쪽)** 과 **감사 인사(마지막 쪽)** 의 담당자 정보에 자동 반영됩니다

---

## 3. PDF로 저장

1. 설정 패널 하단 **「PDF로 저장 (인쇄)」** 클릭
2. 인쇄 대화상자에서 **대상: PDF로 저장** 선택
3. **머리글 및 바닥글** 체크 **해제** (URL·날짜가 찍히지 않게)
4. 저장 → 슬라이드가 한 장씩 깔끔한 PDF로 저장됩니다

> 파일명은 **「ZYXCAD AX 제안서 - 〇〇 (날짜)」** 형식으로 자동 입력됩니다.

---

## 4. 설정 저장 / 불러오기

- **설정 저장** — 입력한 고객사·담당자 정보를 `JSON` 파일로 내보내기
- **불러오기** — 저장한 `JSON`을 다시 불러와 동일하게 채우기
- **초기화** — 모든 입력값을 비우기
- 입력값은 브라우저에도 자동 저장되어, 다시 열면 마지막 상태가 유지됩니다
